# Руководство по камере для мобильных устройств

## Быстрое решение проблем

### 📱 Android (Chrome)
1. **Найдите всплывающий запрос** в верхней части экрана
2. **Нажмите "Разрешить"** для доступа к камере
3. Если запрос не появился:
   - Нажмите на адресную строку
   - Найдите иконку камеры 📷 или замка 🔒
   - Выберите "Разрешить"

### 🍎 iOS (Safari)
1. **Найдите запрос разрешения** в верхней части Safari
2. **Нажмите "Разрешить"**
3. Если не работает:
   - Настройки → Safari → Камера → Разрешить
   - Или Settings → Privacy → Camera → Safari → Включить

## Частые проблемы

### "Cannot read properties of undefined"
**Причина:** Браузер не поддерживает современные API камеры
**Решение:**
- Обновите браузер до последней версии
- Используйте Chrome (Android) или Safari (iOS)
- Избегайте встроенных браузеров приложений

### "Доступ запрещен"
**Причина:** Пользователь заблокировал доступ к камере
**Решение:**
1. Нажмите на адресную строку
2. Найдите иконку камеры и разрешите доступ
3. Обновите страницу

### "Камера занята"
**Причина:** Другое приложение использует камеру
**Решение:**
- Закройте WhatsApp, Instagram, TikTok
- Закройте другие вкладки браузера
- Перезапустите браузер
