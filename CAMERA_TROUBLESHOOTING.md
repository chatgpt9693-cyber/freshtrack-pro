# Устранение проблем с камерой

## Частые проблемы и решения

### 1. "Не удалось получить доступ к камере"

**Причины:**
- Доступ к камере запрещен в браузере
- Камера используется другим приложением
- Отсутствует HTTPS соединение (для продакшена)

**Решения:**
1. **Разрешить доступ в браузере:**
   - Нажмите на иконку 🔒 или 📷 в адресной строке
   - Выберите "Разрешить" для камеры
   - Обновите страницу

2. **Закрыть другие приложения:**
   - Закройте Zoom, Skype, Teams и другие видео-приложения
   - Закройте другие вкладки браузера с камерой

3. **Проверить HTTPS:**
   - Для localhost - работает по HTTP
   - Для продакшена - требуется HTTPS

### 2. "Камера не найдена"

**Решения:**
- Подключите веб-камеру к компьютеру
- Проверьте драйверы камеры
- Перезагрузите браузер
- Попробуйте другой браузер

### 3. "Браузер не поддерживается"

**Рекомендуемые браузеры:**
- ✅ Google Chrome (лучший выбор)
- ✅ Mozilla Firefox
- ✅ Safari (macOS/iOS)
- ✅ Microsoft Edge
- ❌ Internet Explorer (не поддерживается)

### 4. Проблемы на мобильных устройствах

**Android:**
- Используйте Chrome или Firefox
- Разрешите доступ к камере в настройках браузера

**iOS:**
- Используйте Safari или Chrome
- Обновите iOS до последней версии

## Диагностика

Используйте встроенную диагностику в приложении:
1. Откройте сканер штрихкодов
2. Перейдите на вкладку "Справка"
3. Нажмите "Запустить диагностику"
4. Следуйте рекомендациям системы

## Технические требования

- **Браузер:** Chrome 63+, Firefox 60+, Safari 11+, Edge 79+
- **Протокол:** HTTPS (для продакшена)
- **Разрешения:** Доступ к камере
- **Камера:** Любая веб-камера с автофокусом (рекомендуется)
